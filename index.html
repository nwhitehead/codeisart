Danielle Navarro
Selected works, 2019–2025
Presented chronologically.
Each work was generated digitally using a computer program and has many instantiations using different pseudorandom seeds or input files.
Most works are presented abridged based on selections made by me (Nathan).
Zoom and pan on some works are designed to show fine detail.

Danielle Navarro
Rosemary, 2019
License: CC BY 4.0
Repository: https://github.com/djnavarro/series-rosemary
Presented in part (5/12 images)

Paper Cuts

The shapes are simple, but there is real depth. I like the closely spaced curves defining circle edges. I also see a single straight line.

Source code for this work is not available (lost) which makes me obsessed with trying to figure out how it works.

Introduces themes of hurt and damage, common for artists work.

Rainbow Prisms

I can see what is happening a bit more here. There are translucent straight polygons whose vertices match control points of swirly curvy translucent shapes.

Palette is also being algorithmically generated, but I'm not sure how.

I interpret the colors here as LGBT coded, might be over-reaching.

Incantations

This one has a totally different feel, how is it the same program??? But I do see black vertical lines, and the red "text" is swirly translucent shapes that match up with the lines.

This one feels demonic to me, some sort of pact with the devil.

Fracture

I can see evenly spaced dots along lines used as paths for new lines, this gives a clue to how the "texture" works.

Wispy Heart Bright

First appearance of a heart, a recurring theme of the artist.

Heart shape is generated algorithmically (by looking at later works). This one shows a bit more how swirls are contours of a 2D heightmap function.

Pollen, 2020
License: CC0 1.0
Repository: djnavarro/series-pollen
Presented in part (2/76 images)

Pollen 05 214

The detail in these is gorgeous, extreme noise but zoomed out leaving a powdery soft texture.

I like this one because it has interesting square structure somehow linked to the powdery algorithm and not just haphazardly imposed on top. I like the mix of contrained/unconstrained that the points have. I feel like I'm seeing into a divine window of ethereal reality here.

Pollen 04 114

An explosion of points, looks like gravity pulling them down as they float.
